# Audy-Client

## 📑 Description
- 여행계획 최적화 지도 서비스

## 🧱 Dependencies

- pnpm (Package manager)
- Typescript (Programming Language)
- React (SPA)
- Vite (Bundler)
- React-query (Server State Management)
- Vanilla-Extract (Styling)

## 📂 Structure

```
└─ /src
    └─ /apis
    └─ /assets
    └─ /components
    └─ /constants
    └─ /features
    └─ /hooks
    └─ /pages
    └─ /stores
    └─ /query-hooks
    └─ /utils
```
